üìã Manual Testing Instructions

Test the Extension Directly:

  1. Open Chrome DevTools (F12) and go to Console tab
  2. Open the extension and log in
  3. Click "Add New Credential"
  4. Fill in the form:
    • Website: test.com
    • Username: test@example.com
    • Password: testpassword123
  5. Click "Save Credential"
  6. Watch the console for any error messages
  7. If master password modal appears:
    • Try clicking the eye icon to toggle password visibility
    • Enter your master password
    • Click "Continue"

üîß Troubleshooting Steps

If Save Button Doesn't Work:

  1. Check if form validation is preventing submission
  2. Look for JavaScript errors in console
  3. Verify event listeners are attached
  4. Check if button is disabled

If Master Password Modal Issues:

  1. Check if modal HTML elements exist
  2. Verify toggle button event listeners
  3. Check if password field ID matches JavaScript
  4. Look for CSS conflicts hiding elements

If No Action Happens:

  1. Reload the extension in chrome://extensions/
  2. Clear browser cache and reload
  3. Check if background script is running
  4. Verify message passing between popup and background
